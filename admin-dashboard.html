Export Laporan ke Mitra

1) Muat laporan harian Tim bulan ini → 2) Admin rangkum jadi naratif bulanan + Kendala + Rencana Selanjutnya → 3) Generate Laporan Status lengkap (format resmi) → 4) Review/edit langsung di preview → 5) Export.
Disiapkan oleh: — (ubah)
Laporan Tim yang belum tertaut ke Kegiatan RAB (klik "Detail" satu-satu) akan otomatis ditautkan kalau nama Target Capaian yang dipilih Tim persis sama dengan nama Kegiatan RAB ini.

Pilih bulan → "Muat Laporan Harian Bulan Ini" → isi rangkuman → "Simpan & Generate Laporan Status".

Preview di atas bisa diedit langsung (klik teksnya) sebelum di-export — ini tahap Review sebelum dikirim ke Mitra.
